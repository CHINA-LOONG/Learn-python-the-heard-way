Linux/Mac OSX

pwd:打印当前的工作目录

hostname:获取我的计算机的网络名称

mkdir：创建目录

cd：更改当前工作目录

ls:列出目录下的文件和目录

rmdir：删除目录

pushd：push directory

popd:pop directory

cp：复制文件或目录

mv：移动/重命名文件或目录

less：按页查看文件

cat：输出整个文件

xargs：执行参数

find：查找文件

grep：查找文件里面的内容

man：阅读帮助手册

apropos：find what man page is appropriate

env：查看计算机环境

echo：输出内容/参数

export：设置一个新的环境变量

exit：退出终端

sudo：需要超级用户权限
